# LLM 

Система автоматического анализа расшифровок звонков и генерации рекламных креативов на языке клиентов.

## Возможности

- **Автоматический анализ** расшифровок звонков через нейросеть
- **Извлечение инсайтов**: проблемы, возражения, решения, цитаты
- **Генерация креативов**: заголовки и тексты для рекламы
- **Динамический фид**: автоматическое обновление Google Таблицы
- **Кластеризация**: автоматическое тегирование по темам

## Быстрый старт

1. **Установка зависимостей:**
```bash
pip install -r requirements.txt
Настройка API-ключей:

в .env добавить API-ключи

Настроить сервисный аккаунт Google

2) Запуск приложения:

streamlit run app.py

3) Настройка API

3.1) Yandex GPT
Создать аккаунт в Yandex Cloud;
Активировать сервис Yandex GPT;
Получить API-ключ и folder-id;
Добавить в .env.

3.2) Google Sheets API

Создайть проект в Google Cloud Console;
Активируйть Google Sheets API;
Создайть сервисный аккаунт;
Скачать credentials в service_account.json;
Дать доступ к таблице сервисному аккаунту

4) Использование
Загрузить расшифровку звонка (текст или файл);
Запустить анализ - система извлечет инсайты;
Просмотреть креативы - сгенерированные заголовки и тексты;
Сохранить в таблицу - автоматическое пополнение фида.

5) Результаты
Система создает структурированные данные для: A/B тестов рекламных креативов; Анализа клиентских болей; Улучшения продуктов и услуг; Обучения отдела продаж

6) Создание архива

Создать папку `call_analysis_system` и поместите туда все файлы. Затем создать архив.

# Для Linux/Mac
tar -czf call_analysis_system.tar.gz call_analysis_system/

# Для Windows
Используется 7-Zip или встроенный архиватор
